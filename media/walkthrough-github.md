## Backing up to GitHub

GitHub stores a copy of your entire project — including every snapshot — on the internet.

**Why it matters:**
- If your computer breaks or gets lost, your work is safe
- You can access your project from any computer
- Others can collaborate with you

**What "Send to GitHub" does:**
It uploads all your snapshots to GitHub. Your project needs to be connected to a GitHub repo first — Go Git It can set this up automatically if you have the [GitHub CLI](https://cli.github.com) installed.

**The difference between snapshots and GitHub:**
- 📸 **Snapshot** = saved on your computer
- ☁️ **Send to GitHub** = also saved online
