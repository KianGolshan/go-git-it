## Step 1 — Start or open a project

---

### Starting fresh? Build a new project.

Click **🏗️ Build a new project** in the sidebar. A short 3-step wizard walks you through it:

1. **Name your project** — e.g. `My Portfolio` or `Recipe App`
2. **Choose where to save it** — your Documents folder is the default
3. **Connect to GitHub?** — recommended, but you can skip it

Go Git It will:
- Create the folder on your computer
- Set up Git inside it
- Write a `README.md` and `.gitignore` for you
- Make your first snapshot automatically (`🎉 Started My Portfolio`)
- Optionally create a GitHub repo and push everything up

Then VS Code reopens inside your new project. You're ready to build.

---

### Already have a project? Open it.

Click **📂 Open a different project**. Go Git It scans your computer for git projects and shows them in a list — sorted by most recently used.

If your project lives on GitHub but isn't downloaded yet, it appears in the list too. Click it and Go Git It downloads it for you.
