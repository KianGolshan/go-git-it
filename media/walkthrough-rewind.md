## Going back to a previous snapshot

Every snapshot you take is a point in time you can return to.

**The timeline:**
The bottom of the sidebar shows your snapshot history as color-coded dots:
- 🟢 Green = backed up to GitHub
- 🔵 Blue = saved on your computer only
- 🟡 Pulsing yellow = unsaved changes right now

**To go back:**
Click any dot in the timeline. You'll get two options:
- **See what changed** — opens a summary of what was different at that point
- **Go back to this snapshot** — rewinds your project to that exact moment

**Before going back:**
If you have unsaved changes, Go Git It will offer to take a snapshot first so you don't lose anything. You can also choose to go back anyway and discard the current changes.

**Tip:** Take a snapshot before trying anything experimental — it makes rewinding completely safe.
