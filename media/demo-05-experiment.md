## Step 5 — Try something risky with an experiment

---

You want to redesign the whole hero section — but it's currently working and you don't want to break it. **This is exactly what experiments are for.**

Click **🧪 Experiment** in the sidebar. Give it a name:

> `Hero redesign`

> `Dark mode`

> `New navigation`

> `Try Tailwind`

Go Git It creates a completely separate copy of your project to work in. Your main line — everything you've built so far — is frozen exactly as it was.

---

### What changes

The branch badge in the header updates:

> 🧪 Hero redesign

Two new buttons appear in the sidebar:

- **✅ Merge experiment** — bring the changes into your main line
- **🗑️ Abandon experiment** — delete the experiment, return to main

---

### What "separate copy" actually means

It's not literally a copy of all your files. It's a Git branch — the same files, but any changes you make from this point only exist in the experiment. Your main line is untouched until you decide to merge.

---

### You can still take snapshots inside an experiment

In fact, you should. Snapshot freely as you work on the experiment — each one is a save point inside the experiment itself.

---

### What if I forget which branch I'm on?

Always check the badge in the Go Git It header. Green leaf 🌿 means main line. Test tube 🧪 means experiment.
