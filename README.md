# Go Git It

Version control for vibe-coders — no Git jargon, just friendly snapshots.

## What it does

- **Take a snapshot** — save your work at any moment
- **Send to GitHub** — back up everything online
- **Start an experiment** — try risky ideas safely, separate from your main work
- **Visual timeline** — see your history as a friendly storyboard
- **Plain-English errors** — when something goes wrong, we explain it warmly

## Getting started

1. Open a project folder in VS Code
2. Click the **Go Git It** icon in the Activity Bar (left sidebar)
3. Follow the walkthrough — or just click **📸 Take a snapshot** to begin
